Offline AI learning for Android phones and tablets. Download the APK, install it on the device, and keep learning without accounts, cloud sync, or a live internet connection.

Download Android APK Open release page

Use an Android phone or tablet with at least 4 GB RAM. If the APK does not start downloading, open the release page and choose "otic-studio-latest.apk".

Works offline

The app and student data stay on the device. The AI model is added separately from USB or a local server.

Mobile first

The APK installs on Android phones and tablets. Existing student data is preserved during updates.

Windows included

Desktop users can download the Windows zip from the same release page.

Install on Android

  1. Tap the APK download button on the Android device.
  2. Open the downloaded file and allow "Install unknown apps" if Android asks.
  3. Launch Otic Studio. Add the Gemma model file from USB or local storage when prompted.

iPhone and iPad cannot install Android APK files. For classroom distribution, copy the APK to devices by browser download, USB, Bluetooth, or a local school server.
